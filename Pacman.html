Pacman

Using an OpenGL based framework called "SDL", I recreated Pacman to the best of my ability, with only 3 months programming knowledge. considering this was my first attempt at a game I believe i did a great job. Even though it doesn't have path tracing system for the ghosts, it does have a simple collision detection where the ghost randomises it direction, and will never go back on itself.

There are alot of improvements to be made to this game, however as this was my first project, the code is all over the place, but could be improved dramatically with a simple pointer based system.

Go to my Github page to check out the project and all other projects i have made.
